# Entities Standard

An object mapping ~2000 standard HTML entities to their value.

## Install

```sh
npm install --save entities-standard
```

## Usage

```ts
import entities from 'entities-standard';

console.log ( entities );

/*
{
  Aacute: 'Á',
  aacute: 'á',
  Abreve: 'Ă',
  ...and so on...
}
*/
```
